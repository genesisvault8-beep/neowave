What NeoWave Is

NeoWave is a browser-based, peer-to-peer encrypted messenger built by Genesis Group. It uses WebRTC technology to establish direct, encrypted connections between devices — with no central server handling your communications.

NeoWave is hosted on GitHub Pages as a static website. There is no backend server that processes, stores, or logs your conversations.

How Encryption Works

All connections on NeoWave use WebRTC DTLS encryption — the same standard used in modern video calling applications. This encryption is applied automatically by your browser.

Messages and files travel directly between devices. The only component that sees a Wave ID is the PeerJS signaling server, which is used solely to establish the initial connection. Once connected, all data flows peer-to-peer and the signaling server is completely out of the picture.

The signaling server never sees the content of your messages or files — only the Wave ID used to locate your peer.

// 04
🌐
Third-Party Services

NeoWave uses a small number of standard infrastructure services to function — specifically a WebRTC signaling service to establish peer connections, and a font and script delivery network for the interface. These services handle only the technical handshake needed to connect devices.

None of these services have access to your message content, file transfers, or any personal information. Only the temporary Wave ID used to locate your peer is involved in the connection process.

We do not use any analytics, advertising, behavioral tracking, or data monetization services of any kind.

// 05
⚡
Data Retention

When a Wave session ends — when all participants leave or close their browser tab — the session is gone permanently. There is no message history, no chat log, no file cache on any server.

Your browser may temporarily cache static assets (HTML, CSS, JS) via the service worker for offline support. This data never leaves your device and can be cleared at any time through your browser settings.

We retain nothing. There is nothing to delete, request, or export.
